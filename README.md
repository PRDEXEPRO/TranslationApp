# TranslationApp
2 Farklı Uygulama sürümü var;
Web Tabanlı Translate uygulaması ve Standart Translate Uygulaması.
Web Tabanlı Translate uygulamasını çalıştırmak için, cmd üzerinden dotnet run komutunu çalıştırın ve http://localhost:5055 adresine gidin.
Standart Translate uygulamasını çalıştırmak için, cmd üzerinden dotnet run komutunu çalıştırın.
